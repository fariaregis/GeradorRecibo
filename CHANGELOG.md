# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2025-12-09

### Adicionado
- Sistema de cadastro de empresas multi-tenant
- Cadastro de clientes (PF e PJ)
- Geração de recibos em PDF
- Categorização de serviços prestados
- Conversão automática de valores por extenso
- Formatação automática de CPF/CNPJ e telefones
- Sistema de login por empresa
- Configuração de empresa padrão
- Histórico completo de recibos
- Visualização de recibos emitidos
- Interface maximizada para melhor usabilidade

### Recursos
- Layout profissional de recibos
- Validação de CPF/CNPJ
- Máscaras de entrada automáticas
- Abertura automática do PDF ao gerar recibo
- Texto jurídico de quitação nos recibos
